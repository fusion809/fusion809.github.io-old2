Basic Usage

Table 4: Basic Usage of pacman
Action Command1 Description
Build package from source {% include Code/codeu.html line1="makepkg" %} Build package from source, using instructions in the PKGBUILD in the present working directory (PWD).
Install build dependencies {% include Code/codeu.html line1="makepkg -s" %} Install the required build dependencies for package's whose PKGBUILD is in the PWD.
Install package {% include Code/coder.html line1="pacman -S $package" %} Install the target package ($package) and all runtime dependencies.
Remove package {% include Code/coder.html line1="pacman -Rs $package" %} Remove the target package ($package) and all runtime dependencies not required by other installed packages.
Search for keyword {% include Code/coder.html line1="pacman -Ss $package" %} Search package names and descriptions for provided regex.
Upgrade all packages {% include Code/coder.html line1="pacman -Syu" %} Upgrade all installed packages present in the pacman repositories.
1: Note: in this table I use the short-hand notation $package to refer to the target package(s) that the user wishes to perform the mentioned action on.
