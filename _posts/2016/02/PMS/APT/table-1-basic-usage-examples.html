### Basic Usage
Table 1: Basic Usage of APT
Action Command1 Description
Add a repository {% include Code/coder.html line1="add-apt-repository $repository" %} Install required build dependencies for the target package ($package).
Install build dependencies {% include Code/coder.html line1="apt-get build-dep $package" %} Install required build dependencies for the target package ($package).
Install package {% include Code/coder.html line1="apt-get install $package" %} Install the target package ($package) and all runtime dependencies.
Remove package {% include Code/coder.html line1="apt-get remove $package" %} Remove the target package ($package) and all runtime dependencies not required by other installed packages.
Search package {% include Code/coder.html line1="apt-cache search $package" %} Search for package(s).
1: Note: in this table I use the short-hand notations $package and $repository to refer to the target package(s) or repository, respectively, that the user wishes to perform the mentioned action on.
