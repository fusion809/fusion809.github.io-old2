NAME

php-pear-lib-r1.eclass - Provides means for an easy installation of PEAR-based libraries.

DESCRIPTION

This class provides means for an easy installation of PEAR-based libraries, such as Creole, Jargon, Phing etc., while retaining the functionality to put the libraries into version-dependant directories.

FUNCTIONS

php-pear-lib-r1_pkg_setup
Adds required PEAR channel if necessary
php-pear-lib-r1_src_install
Takes care of standard install for PEAR-based libraries.
