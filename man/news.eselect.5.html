NAME

news.eselect - The GLEP 42 newsreader module for Gentoo's eselect

SYNOPSIS

eselect news [help|usage|version]
 
eselect news list
 
eselect news read [--mbox|--quiet|--raw] [item...]
 
eselect news unread [item...]
 
eselect news purge
 
eselect news count [new|all]

DESCRIPTION

eselect is Gentoo's configuration and management tool. It features modules that care for the individual administrative tasks.

ACTION: LIST

eselect news list
 
List all relevant Gentoo news items. Unread items are flagged by an N next to their date.
 
# eselect news list
 
News items:
 

[1] 2009-04-06 Migration from teTeX to TeXLive
[2] 2009-04-06 Migration to X.org Server 1.5
[3] N 2009-04-18 Generation 1 Java Setup Deprecated

ACTION: READ

eselect news read [--mbox|--quiet|--raw] [item...]
 
Read news item(s), selected by their item number(s) from the list action. Special items new and all select all unread items or all items, respectively. Default is to read all unread news, if no item is specified. With options --mbox or --raw, output the item(s) in mbox format or in raw format, respectively. Option --quiet or --silent suppresses output, but items are still marked as read.

ACTION: UNREAD

eselect news unread [item...]
 
Mark news item(s) as unread again. Items are selected by their item number(s) from the list action. Special item all selects all items.

ACTION: PURGE

eselect news purge
 
Purge all read news item(s).

ACTION: COUNT

eselect news count [new|all]
 
Display number of unread news items (default) or of all news items, if option all is given.
