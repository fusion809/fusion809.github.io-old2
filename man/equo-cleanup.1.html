NAME

equo-cleanup - remove downloaded packages and clean temp. directories

SYNOPSIS

equo cleanup [-h]

INTRODUCTION

Remove downloaded packages and clean temporary directories.

OPTIONS

"equo cleanup" supports the following options which alters its behaviour.

OPTIONAL ARGUMENTS

--help
show this help message and exit
