NAME

nvidia-driver.eclass - Provide useful messages for nvidia-drivers based on currently installed Nvidia card

DESCRIPTION

Provide useful messages for nvidia-drivers based on currently installed Nvidia card. It inherits versionator.

FUNCTIONS

nvidia-driver-get-card
Retrieve the PCI device ID for each Nvidia video card you have
nvidia-driver-check-warning
Prints out a warning if the driver does not work w/ the installed video card
