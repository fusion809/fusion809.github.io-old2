NAME

gnome-python-common-r1.eclass - Common functionality for building gnome-python* bindings

DESCRIPTION

This eclass provides python-r1 support for the GNOME2 library Python bindings.

ECLASS VARIABLES

G_PY_BINDINGS ?= ${PN%-python}
The actual '--enable-<binding>' name. If multiple bindings are to be enabled, must be an array.
EXAMPLES
The set of example files to be installed if the 'examples' USE flag is set.
