NAME

fox.eclass - Functionality required the FOX Toolkit and it's applications

DESCRIPTION

This eclass allows building SLOT-able FOX Toolkit installations (x11-libs/fox: headers, libs, and docs), which are by design parallel-installable, while installing only one version of the utils (dev-util/reswrap) and apps (app-editors/adie, sci-calculators/calculator, x11-misc/pathfinder, and x11-misc/shutterbug).
 
Version numbering follows the kernel-style odd-even minor version designation. Even-number minor versions are API stable, which patch releases aimed mostly at the library; apps generally won't need to be bumped for a patch release.
 
Odd-number versions are development branches with their own SLOT and are API unstable; changes are made to the apps, and likely need to be bumped together with the library.
 
Here are sample [R]DEPENDs for the fox apps 1.6: 'x11-libs/fox:1.6' 1.7: '~x11-libs/fox-${PV}'
 
EAPI phase trickery borrowed from enlightenment.eclass

ECLASS VARIABLES

FOX_PV ?= ${PV}
The version of the FOX Toolkit provided or required by the package
FOXCONF
Set this to add additional configuration options during src_configure
