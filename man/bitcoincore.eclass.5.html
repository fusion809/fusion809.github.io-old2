NAME

bitcoincore.eclass - common code for Bitcoin Core ebuilds

DESCRIPTION

This eclass is used in Bitcoin Core ebuilds (bitcoin-qt, bitcoind, libbitcoinconsensus) to provide a single common place for the common ebuild stuff.
 
The eclass provides all common dependencies as well as common use flags.

ECLASS VARIABLES

BITCOINCORE_COMMITHASH
Set this variable before the inherit line, to the upstream commit hash.
BITCOINCORE_IUSE
Set this variable before the inherit line, to the USE flags supported.
BITCOINCORE_LJR_DATE
Set this variable before the inherit line, to the datestamp of the ljr patchset.
BITCOINCORE_POLICY_PATCHES
Set this variable before the inherit line, to a space-delimited list of supported policies.
