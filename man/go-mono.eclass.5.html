NAME

go-mono.eclass - Common functionality for go-mono.org apps

DESCRIPTION

Common functionality needed by all go-mono.org apps.

FUNCTIONS

go-mono_src_unpack
Runs default()
go-mono_src_prepare
Runs autopatch from base.eclass, if PATCHES is set.
go-mono_src_configure
Runs econf, disabling static libraries and dependency-tracking.
go-mono_src_compile
Runs emake.
go-mono_src_install
Rune emake, installs common doc files, if DOCS is set, installs those. Gets rid of .la files.

ECLASS VARIABLES

DOCS
Insert path of docs you want installed. If more than one, consider using an array.
