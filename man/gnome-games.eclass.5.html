NAME

gnome-games.eclass - An eclass to build gnome-games.

DESCRIPTION

An eclass to build gnome-games using proper phases from gnome2 and games eclasses.

FUNCTIONS

gnome-games_pkg_setup
Set proper phase defaults
gnome-games_src_prepare
Set proper phase defaults
gnome-games_src_configure
Set proper phase defaults, relying on gnome2_src_configure and passing extra arguments from egamesconf (games.eclass)
gnome-games_src_compile
Set proper phase defaults
gnome-games_src_install
Set proper phase defaults
gnome-games_pkg_preinst
Set proper phase defaults
gnome-games_pkg_postinst
Set proper phase defaults
gnome-games_pkg_postrm
Set proper phase defaults
