NAME

aspell-dict.eclass - An eclass to streamline the construction of ebuilds for new aspell dicts

DESCRIPTION

The aspell-dict eclass is designed to streamline the construction of ebuilds for the new aspell dictionaries (from gnu.org) which support aspell-0.50. Support for aspell-0.60 has been added by Sergey Ulanov.

FUNCTIONS

aspell-dict_src_compile
The aspell-dict src_compile function which is exported.
aspell-dict_src_install
The aspell-dict src_install function which is exported.

ECLASS VARIABLES

ASPELL_LANG (REQUIRED)
Which language is the dictionary for? It's used for the DESCRIPTION of the package.
ASPOSTFIX (REQUIRED)
What major version of aspell is this dictionary for?
