NAME

equo - The Ingenuous Entropy Package Manager.

SYNOPSIS

equo [command] [command options]

INTRODUCTION

Equo is the command-line frontend of the Entropy Package Manager Library. Installing, Removing and taking care of your system is easier than ever. There is nothing much left to say, just try it out yourself!

USAGE

Equo has a modular design, commands can be added and removed in a pluggable way. There are however a set of built-in commands that are shipped with the default Equo distribution.

OPTIONS

--help
Show Equo Help, listing all the runtime available commands.
