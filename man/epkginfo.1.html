EPKGINFO(1) EPKGINFO(1)

NAME

epkginfo - Displays metadata information from packages in portage
 

SYNOPSIS

epkginfo [options] package name or atom
 

DESCRIPTION

Epkginfo is a shortcut to equery meta. For all available options and more examples, see man equery.
