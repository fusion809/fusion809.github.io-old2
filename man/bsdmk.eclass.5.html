NAME

bsdmk.eclass - Some functions for BSDmake

FUNCTIONS

append-opt < options >
append options to enable or disable features
mkmake [ options ]
calls bsd-make command with the given options, passing ${mymakeopts} to enable ports to useflags bridge.
mkinstall [ options ]
Calls "bsd-make install" with the given options, passing ${mamakeopts} to enable ports to useflags bridge
dummy_mk < dirnames >
removes the specified subdirectories and creates a dummy makefile in them useful to remove the need for "minimal" patches
bsdmk_src_compile
The bsdmk src_compile function, which is exported
bsdmk_src_install
The bsdmk src_install function, which is exported

ECLASS VARIABLES

mymakeopts
Options for bsd-make
