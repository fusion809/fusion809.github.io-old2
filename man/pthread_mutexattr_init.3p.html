NAME

pthread_mutexattr_init — initialize the mutex attributes object

SYNOPSIS


#include <pthread.h>
int pthread_mutexattr_init(pthread_mutexattr_t *attr);

DESCRIPTION

Refer to pthread_mutexattr_destroy().
