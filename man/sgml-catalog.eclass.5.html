NAME

sgml-catalog.eclass - Functions for installing SGML catalogs

FUNCTIONS

sgml-catalog_cat_include <centralized catalog> <ordinary catalog>
Appends a catalog pair to the SGML_TOINSTALL array.
sgml-catalog_cat_doinstall <centralized catalog> <ordinary catalog>
Adds an ordinary catalog to a centralized catalog.
sgml-catalog_cat_doremove <centralized catalog> <ordinary catalog>
Removes an ordinary catalog from a centralized catalog.

ECLASS VARIABLES

SGML_TOINSTALL = ()
An array of catalogs, arranged in pairs. Each pair consists of a centralized catalog followed by an ordinary catalog.
