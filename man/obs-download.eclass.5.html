NAME

obs-download.eclass - Simplifies downloading from openSUSE Build Service.

DESCRIPTION

This eclass constructs OBS_URI based on provided project in openSUSE Build Service and package name. It can be used by packages/eclasses to download actual files.
 
All you need to do in order to use it is set OBS_PROJECT and OBS_PACKAGE and inherit this eclass. It will provide OBS_URI in return which you will prepend to your files and use it in SRC_URI. Alternatively you can just set OPENSUSE_RELEASE and OBS_PACKAGE and it will give you back OBS_URI for downloading files from obs project corresponding to the specified openSUSE release.

ECLASS VARIABLES

OPENSUSE_RELEASE
From which openSUSE realease to take files. Eg.: 12.1, 12.2, Factory
OBS_PROJECT
In which obs project pakage is. This variable does not have to be set and is overriden, if OPENSUSE_RELEASE is provided.
OBS_PACKAGE
Name of the package we want to take files from. By default taken from ${PN}.
