NAME

php-pear-r1.eclass - Provides means for an easy installation of PEAR packages.

DESCRIPTION

This eclass provides means for an easy installation of PEAR packages. For more information on PEAR, see http://pear.php.net/ Note that this eclass doesn't handle dependencies of PEAR packages on purpose; please use (R)DEPEND to define them correctly!

FUNCTIONS

php-pear-lib-r1_pkg_setup
Adds required PEAR channel if necessary
php-pear-r1_src_install
Takes care of standard install for PEAR packages.

ECLASS VARIABLES

PHP_PEAR_PKG_NAME = "${PN/PEAR-/}"
Set this if the the PEAR package name differs from ${PN/PEAR-/} (generally shouldn't be the case).
PEAR_PV
Set in ebuild if the eclass ${PV} mangling breaks SRC_URI for alpha/beta/rc versions
